# Notes
## Number row characters
* `!` Exclamation point, Bang
* `@` At
* `#` Hashtag, Hash, Pound
* `$` Dollar
* `%` Percent, 
* `^` Caret, Circumflex, Exponent operator
* `&` Ampersand, And
* `*` Asterisk
* `(` Open/Left Parenthesis
* `)` Close/Right Parenthesis
